# オーナーガイド — 予備知識ゼロで読める、あなたの役割のすべて

このページは、**アプリを作る人ではなく、アプリの持ち主(オーナー)のための**説明書です。プログラミングやサーバーの知識は一切仮定しません。English version: [owner-guide.md](owner-guide.md).

あなたのアプリは、あなた自身か、あなたが指示したコーディングエージェント(Claude Code など)が作り、公開作業もエージェントが代行できます。その過程で**あなたにしかできないことは 3 つだけ**です:

1. **AI の利用料を払う口座(API キー)を用意する** — 10〜15 分
2. **アプリを動かす場所(ホスティング)のアカウントを用意する** — 10 分
3. **決める**: 月にいくらまで使ってよいか・誰を招待するか

それぞれ「そもそも何なのか」から説明します。

---

## 1. API キーとは何か(「Anthropic キー」の正体)

AI(Claude)は、あなたのパソコンの中ではなく、提供元(Anthropic 社や Amazon)のコンピューターで動いています。アプリが AI に質問を送るたびに、提供元は「これは誰からの依頼で、誰に請求すればいいのか」を知る必要があります。その身元証明に使われる長い文字列(例: `sk-ant-api03-...`)が **API キー**です。

つまり API キーとは、**「あなたの支払いにつながる、AI 利用のための鍵」**です。だからこそ:

- キーを持つ人は誰でも**あなたのお金で** AI を使えます。**絶対に SNS やチャットに貼らない**でください
- sekimori の存在理由がまさにこれです: キーをサーバーの中だけに置き、アプリの利用者には**招待トークン**(上限つきの通行手形)だけを配ります。キーそのものは誰にも渡りません

### 入手方法は 2 通りあります(どちらか 1 つでよい)

**方法 A: Anthropic から直接(標準的な方法)**

1. [console.anthropic.com](https://console.anthropic.com) でアカウントを作る(メールアドレスがあればよい)
2. 支払い方法を登録し、**前払いクレジットを購入**する(最低 $5 程度から。使った分だけ減る方式で、勝手に追加請求されません)
3. 「API Keys」のページで「Create Key」を押し、表示された `sk-ant-...` を控える(**この画面でしか表示されません**)
4. さらに安心のため、Anthropic の管理画面にも**月あたりの利用上限**を設定できます。sekimori の上限と二重の守りになります

費用の目安: 小さなアプリを友人数人に使わせる程度なら、**月に数十円〜数百円**($0.5〜$5)に収まることがほとんどです。上限は sekimori が強制するので、想定外の請求は起きません。

**方法 B: Amazon Bedrock 経由(AWS のクレジットを持っている人向け)**

AWS(Amazon のクラウド)のアカウント登録でもらえる**無料クレジットは、Bedrock という AI サービスの利用料にも充当できます**。Bedrock 上でも Claude が動いており、sekimori は Bedrock にも対応しています(注意点が 1 つ、下記)。

1. [AWS アカウント](https://aws.amazon.com/jp/)を作る(既にあればそれでよい)
2. AWS の管理画面で「Amazon Bedrock」を開き、**モデルアクセス**のページで Anthropic Claude の利用を申請する(フォームに用途を書くだけ。通常すぐ承認されます)
3. Bedrock の「API keys」のページで **API キーを発行**する(これが方法 A のキーに相当します)
4. クレジットの適用条件(対象サービス・期限)は AWS の画面で必ず確認してください

**注意点**: Bedrock 経由では、現在の sekimori は「文字がぽつぽつと流れるように表示される」方式(ストリーミング)が使えず、**返答が一括で表示される方式のみ**です。チャットとしては問題なく使えますが、体感は少し変わります(対応予定はロードマップ参照)。

> どちらを選ぶかの目安: **AWS クレジットが手元にあるなら方法 B**(追加出費ゼロで始められる)。**なければ方法 A**(手順が短く、流れる表示も使える)。

## 2. ホスティング(アカウント)とは何か

あなたのパソコンでアプリを動かしても、パソコンを閉じれば止まりますし、他人はアクセスできません。**インターネット上に常時つながった、借り物のコンピューターでプログラムを動かし続けてくれるサービス**をホスティングと呼びます。「ホスティングアカウントを用意する」とは、そういうサービス(例: [Fly.io](https://fly.io)、[Railway](https://railway.com) など)に**メールアドレスで会員登録し、支払いカードを登録する**ことです。

- 費用の目安: この規模のアプリなら**月 $0〜5** 程度。無料枠に収まることも多いです
- どのサービスを選ぶか・その後の設定はすべて**エージェントに任せてかまいません**。エージェントに「どのサービスのアカウントを作ればいい?」と聞けば、手順を案内してくれます
- ここで登録するカードは「アプリを置く場所代」、方法 A/B のキーは「AI の利用料」で、**別々の支払い**です

## 3. あなたが決めること(エージェントに聞かれたら答えること)

| 質問 | 意味 | 迷ったら |
|---|---|---|
| 月の上限はいくら? | 全員合計で AI に使ってよい月額。超えると自動停止(追加請求なし) | $5〜30 |
| 1 人あたり 1 日いくら? | 招待した 1 人が 1 日に使える額。超えるとその人だけ翌日まで停止 | $0.5〜1 |
| 誰を招待する? | 招待コード(トークン)を渡す相手。1 人ずつ即時に取り消せます | — |

## 4. キーを安全にエージェントへ渡すには

API キーは「秘密の値」です。渡し方だけ注意してください:

- **よい渡し方**: ホスティングサービスの「環境変数」「Secrets」と呼ばれる設定欄に自分で貼り付ける(エージェントが場所を案内します)。または、エージェントが用意した安全な設定手順に従う
- **避ける渡し方**: 公開される場所(SNS、共有されるチャットログ、アプリのコードの中)に貼ること
- 万一漏れた場合は、発行元の管理画面(Anthropic Console / AWS)でそのキーを**無効化**すれば、それ以降は使えなくなります。sekimori 側の上限があるため、漏えいに気づくまでの被害も上限額までで止まります

## 5. 設置が終わったら、何を確認すればいいか

エージェントは設置後に、次のような**保護の要約**をあなたに報告します(sekimori の自己診断機能 `doctor` が自動生成する内容に基づきます):

> - 使えるモデル: claude-haiku-4-5
> - 月の上限: 全員合計 $30 / 1 人 1 日 $0.5
> - 1 人あたり毎分 10 回まで
> - 会話の中身は記録されない
> - API キーは誰の画面にも出ない

この要約に**あなたが決めた数字が入っているか**だけ確認してください。それが監査の代わりになります。数字が違えば、エージェントに「上限を $X にして」と言い直せば済みます。

## よくある質問

**Q. AI の請求が青天井にならないか心配です。**
A. 二重に守られます。(1) sekimori が上限額で自動遮断します(遮断は「壊れた」のではなく機能です)。(2) 方法 A なら前払い式なので、購入額以上は物理的に使えません。方法 B でも AWS 側で予算アラートを設定できます。

**Q. AWS Bedrock の無料クレジットは使えますか?**
A. 使えます(方法 B)。sekimori は Bedrock の API キー認証に対応しています。制約は 1 つ: 現状は返答の「流れる表示」(ストリーミング)に非対応で、一括表示になります。

**Q. 招待した友人が使いすぎたら?**
A. その人だけが日次上限で自動停止し、翌日(UTC)に自動再開します。全体の月上限に達した場合は全員停止し、翌月に再開します。いずれも追加請求は発生しません。

**Q. アプリを止めたくなったら?**
A. エージェントに「止めて」と言えば、ホスティング側でアプリを停止できます。API キーを無効化すれば AI 利用も完全に止まります。

**Q. このガイドの英語版・技術的な詳細は?**
A. 英語版は [owner-guide.md](owner-guide.md)。技術的な仕組みは [README](../README.md)(開発者向け)と [AGENTS.md](../AGENTS.md)(エージェント向け)にあります。あなたが読む必要はありません。
