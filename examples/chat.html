A zero-dependency demo page that just POSTs /v1/messages (stream: true) through sekimori using an invite token. Because of CORS, don't open it via file:// - serve it from a simple HTTP server instead (e.g. run python3 -m http.server 8000 in this directory and add the serving origin to sekimori's cors.allowedOrigins).
