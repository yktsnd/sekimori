A zero-dependency demo page that POSTs /v1/messages through sekimori using an invite token. Because of CORS, don't open it via file:// - serve it from a simple HTTP server instead (for example, run python3 -m http.server 8000, or py -m http.server 8000 on Windows, in this directory and add the serving origin to sekimori's cors.allowedOrigins). Set CONFIG.stream to false for a Bedrock upstream. For a non-local deployment, serve both this page and the gateway over HTTPS to avoid mixed-content blocking.

This token is saved only in this browser. On a shared device, use “Change token” when you finish.
